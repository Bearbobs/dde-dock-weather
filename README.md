Deepin dock weather plugin 
==========================

This is a weather plugin for deepin-dock, with API from [openweathermap](https://openweathermap.org).

![alt](preview.png)
![alt](preview1.png)
![alt](preview2.png)

### Installation Guide ###
Compiling using Qt Creator or using the following command:
```
mkdir build
cd build
qmake ../source
make
```

And than install
```
sudo make install
```
